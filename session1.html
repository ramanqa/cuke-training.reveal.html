Given-When-Then explained


  • Scenario - title/description for the scenario example
  • Given - conditions necessary for the system to attain test base state and to set a context
  • When - actions performed and data entered that invokes system's response
  • Then - behavior or system's response for performed actions

Scenario: title/test objective/behavior description
  Given some initial context (the givens),
  When an event occurs,
  Then ensure some outcomes.

What's in a Story

The sturcture of a story


Title: one line describing the story

Narrative:
As a [ROLE]
I want [FEATURE/FUNCTION]
So that [BENEFIT]

Acceptance Criteria:
  • - criteria 1

Story - ATM Transaction


Story: Account holder withdraws cash from ATM

Narrative: As an Account Holder
   I want to withdraw cash from an ATM
   So that I can get money when bank is closed

Acceptance Criteria:
  • - Account holder can withdraw cash when account has sufficient funds
  • - Account holder can not withdraw cash when account has insufficient funds
  • - Account holder can not withdraw cash when ATM card has expired

Specification by Example

Specification by Example

Feature: Account Holder withdraws money from ATM

  As an Account Holder
  I want to withdraw cash from an ATM
  So that I can get money when bank is closed

  Example: Account holder can withdraw cash when account has
     sufficient funds
    Given account balance is Rs.100
    Given ATM card is valid
    Given ATM machine has more than Rs.100 in cash
    When Account Holder requests Rs.50
    Then ATM machine dispeses Rs.50
    Then ATM card is returned
    Then new account balance is Rs.50

  Example: Account holder can not withdraw cash when account has
     insufficient funds
    Given account balance is Rs.10
    Given ATM card is valid
    Given ATM machine has more than Rs.100 in cash
    When Account Holder requests Rs.50
    Then ATM machine does not dispense money
    Then ATM card is returned
    Then ATM machine shows error message 'insufficient
      account balance'
    Then new account balance is Rs.10

  Example: Account holder can not withdraw cash when ATM card has
     expired
    Given account balance is Rs.100
    Given ATM card has expired
    Given ATM machine has more than Rs.100 in cash
    When Account Holder requests Rs.50
    Then ATM machine does not dispense money
    Then ATM card is returned
    Then ATM machine shows error message 'ATM card expired'
    Then new account balance is Rs.100

Scenarios in Story should...


  • be examples that detail the acceptance criteria
  • expand feature behavior understanding into context (Given), actions (When) and outcomes (Then)
  • bridge feature understanding between all sdlc team members
  • be executeable
